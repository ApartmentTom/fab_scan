# Flesh and Blood Card Scanner

A Python-based card scanning and identification system for Flesh and Blood TCG cards.

## Features

- Card detection and image preprocessing
- Card identification from images
- Support for various image formats

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python scan_card.py <image_path>
```

## Test Images

The repository includes test images of Flesh and Blood cards for testing the scanner.
